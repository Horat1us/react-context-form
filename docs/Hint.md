# <Hint /> Component
This component is represents error hint for [<Form />](./Form.md).
It requires [context](../src/Form/FormGroupContext.ts) which is provided by [<FormGroup/>](./FormGroup.md).

### Usage

```tsx
<Form {...FormProps}>
    <FormGroup {...FormGroupProps}>
        <Hint  {...HTMLDivElementProps} />
    </FormGroup>
</Form>
```
