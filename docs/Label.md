# <Label /> Component
This component is represents label for [<Form />](./Form.md).
It requires [context](../src/Form/FormGroupContext.ts) which is provided by [<FormGroup/>](./FormGroup.md).

### Usage

```tsx
<Form {...FormProps}>
    <FormGroup {...FormGroupProps}>
        <Label  {...HTMLLabelElementProps} />
    </FormGroup>
</Form>
```
